Gulp
 - compressão de imagens
 - compressão de javascripts
 - compressão e compilação de scss(sass) para css
 - compressão de html
 - clean (remover tudo?)
 - scaffold (criar os 'dist' versão para distribuição / produção)
 - browser sync
 - watch (cada alteração que acontecer, não precisa dar refresh , pastas pré configuradas)
 - deploy (build para produção)

Bower
 - package manager (igual ao NPM, porém mais voltado para o front end)
 - depois de instalado, a forma de importar frameworks é:
 - vai no bowerrc para determinar onde os pacotes serão instalados
$ bower install <nome-do-pacote> (igual ao npm)
$ bower install jquery
 - vai no bowerrc para determinar onde os pacotes serão instalados
 - para saber os nomes dos paotes, busca no site bower.io/search
